Scenarios

State

The brush only streams pressure during an active session, and battery updates on connect (not live). Sector/zone isn't decoded on-device. Design within those limits and the lambda translation is 1:1.

Rendering

Font — pick a preset, or type ANY Google Font name in Custom and it's fetched live for preview. The same name maps to gfonts://<name> on-device, so whatever previews here is reproducible in the firmware. The readout below names the family actually rendering (and warns if a name didn't load).
